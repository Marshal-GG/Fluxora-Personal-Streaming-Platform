# API Contracts

> **Category:** API  
> **Status:** Active — Sourced from Planning Session (2026-04-27)

---

## API Style

**REST over HTTP** — FastAPI (Python)  
**Streaming:** HLS over HTTP (`.m3u8` + `.ts`)  
**Real-time:** WebSocket for signaling and status  

---

## Base URLs

| Environment | URL |
|-------------|-----|
| Local (LAN) | `http://{server_local_ip}:8000` |
| Internet (TURN) | Tunneled through WebRTC data channel |

---

## Authentication

All endpoints (except `/info` and `/auth/*`) require:
```
Authorization: Bearer {auth_token}
```

Token is issued after successful client pairing.

---

## Endpoints

### `GET /api/v1/info`
**Description:** Returns server identity — used during discovery.  
**Auth:** None required.  
**Status:** ✅ Implemented

**Response:**
```json
{
  "server_name": "My Fluxora Server",
  "version": "1.0.0",
  "tier": "plus"
}
```

---

### `POST /auth/request-pair`
**Description:** Client requests to be paired with the server.

**Request:**
```json
{
  "device_name": "Pixel 8 Pro",
  "platform": "android"
}
```

**Response (pending):**
```json
{
  "client_id": "uuid",
  "status": "pending_approval"
}
```

---

### `GET /auth/status/{client_id}`
**Description:** Poll for pairing approval.

**Response:**
```json
{
  "status": "approved",
  "auth_token": "eyJ..."
}
```

---

### `GET /files`
**Description:** Browse the server file system or a specific directory.

**Query Params:**
| Param | Type | Description |
|-------|------|-------------|
| `path` | string | Directory path (default: root library dirs) |

**Response:**
```json
{
  "path": "/media/movies",
  "items": [
    { "name": "Inception.mkv", "type": "file", "size_bytes": 8000000000, "id": "uuid" },
    { "name": "Action", "type": "directory" }
  ]
}
```

---

### `GET /library`
**Description:** List all libraries.

**Response:**
```json
[
  { "id": "uuid", "name": "Movies", "type": "movies", "file_count": 142 }
]
```

---

### `POST /library/scan`
**Description:** Trigger a library re-scan.

**Request:** (optional `library_id`, else scans all)
```json
{ "library_id": "uuid" }
```

**Response:**
```json
{ "status": "scanning", "job_id": "uuid" }
```

---

### `GET /stream/{file_id}`
**Description:** Returns HLS playlist URL for streaming a file.

**Response:**
```json
{
  "stream_url": "http://192.168.1.10:8000/hls/abc123/playlist.m3u8",
  "session_id": "uuid"
}
```

---

### `GET /hls/{session_id}/playlist.m3u8`
**Description:** Serves the HLS master playlist (generated by FFmpeg).  
Content-Type: `application/vnd.apple.mpegurl`

---

### `GET /hls/{session_id}/{segment}.ts`
**Description:** Serves individual HLS video segment.  
Content-Type: `video/MP2T`

---

### `WebSocket /ws/signal`
**Description:** WebRTC signaling channel for internet connections.

**Messages:**
```json
// Client → Server (offer)
{ "type": "offer", "sdp": "...", "client_id": "uuid" }

// Server → Client (answer)
{ "type": "answer", "sdp": "..." }

// ICE candidates (both directions)
{ "type": "ice-candidate", "candidate": "..." }
```

---

### `WebSocket /ws/status`
**Description:** Real-time stream heartbeat — updates playback position.

**Client sends every 5s:**
```json
{ "session_id": "uuid", "progress_sec": 342.5 }
```

## Error Codes

| Code | Meaning |
|------|---------|
| 400 | Bad Request — missing or invalid params |
| 401 | Unauthorized — missing or invalid token |
| 403 | Forbidden — client not trusted / tier limit |
| 404 | Not Found — file or library not found |
| 429 | Too Many Requests — stream concurrency limit |
| 500 | Internal Server Error |
| 503 | FFmpeg unavailable |
